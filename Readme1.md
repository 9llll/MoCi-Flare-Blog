================================================================================
                    🌿 莫辞の资源库 - 完整功能清单
================================================================================

一个基于 Cloudflare Workers + D1 + R2 + KV 的轻量级博客与资源管理系统。


================================================================================
                      📝 文章管理功能
================================================================================

  ✅ 文章列表展示（卡片式布局）
  ✅ 文章详情页（富文本内容渲染）
  ✅ 文章分类（父分类 + 子分类）
  ✅ 文章归档（按年份分组）
  ✅ 文章搜索（标题/摘要/内容/分类）
  ✅ 相关文章推荐（同分类）
  ✅ 上一篇 / 下一篇导航
  ✅ 文章目录自动生成（TOC）
  ✅ 文章封面图支持
  ✅ 文章摘要自动截取
  ✅ 文章排序（自定义排序）
  ✅ 文章创建/编辑/删除（管理后台）
  ✅ Quill 富文本编辑器（图片上传/粘贴）
  ✅ 代码高亮（Highlight.js）


================================================================================
                      📦 资源下载功能
================================================================================

  ✅ 三种资源类型：
     ├── 🌐 公开资源（无需验证，直接下载）
     ├── 🔒 密码资源（输入密码后下载）
     └── 💰 付费资源（扫码支付后下载）

  ✅ 资源多链接支持（一篇文章多个资源）
  ✅ 资源缩略图展示
  ✅ 资源标题/描述/按钮文字自定义
  ✅ 付费资源：
     ├── 自定义价格
     ├── 收款二维码上传
     ├── 订单系统（待确认/已完成）
     └── 支付确认后邮件发送下载链接
  ✅ 密码资源：
     ├── SHA-256 哈希存储（不存明文）
     ├── 前端哈希验证（密码不经过网络传输）
     └── 密码错误提示（震动动画）


================================================================================
                      🔒 安全跳转功能
================================================================================

  ✅ AES-GCM 加密 URL（银行级加密）
  ✅ go.html 免责声明页面（用户阅读后确认跳转）
  ✅ 真实 URL 永不暴露（URL 栏只显示加密 token）
  ✅ 用户点击"同意" → 新窗口打开资源
  ✅ 用户点击"不同意" → 返回文章页
  ✅ 键盘快捷键（Enter 同意 / ESC 不同意）
  ✅ 微信/QQ 检测 → 自动跳转浏览器提示页
  ✅ go-browser.html 提示用户用浏览器打开
